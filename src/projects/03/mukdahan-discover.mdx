##### 📝 Mukdahan-discover Project
- เป็นโปเจกต์เพื่อการโปรโมทการท่องเที่ยวในจังหวัดมุกดาหาร
- เรียนรู้การทำเว็บด้วย Next.JS
- เรียนรู้การเขียน UI ด้วย Styled-components
- เรียนรู้การทำเว็บ Animation

##### 📚 Using :
- Next.JS => ทำ Front-end
- Styled-components => จัด Layout ทำ UI
- Swiper => ทำ Image slider
- AOS => ทำ Animation
- Remixicon => Icons
- React-share => ทำปุ่มแชร์
- React-scroll => ทำ Smooth scrolling
- Vercel => Deploy

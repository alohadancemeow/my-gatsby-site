##### 📝 Chatroom-demo
- เว็บเแชทแบบเรียลไทม์ ที่ไม่มีการเก็บขู้อมูล ประวัติการพูดคุย
- เรียนรู้การทำเว็บแชทด้วย Socket.io
- เรียนรู้การทำระบบ login ด้วย jsonwebtoken และเข้ารหัสด้วย bcrypt
- เรียนรู้การทำ Back-end ด้วย Node.JS + MongoDB

##### 📚 Using :
- React.JS => ทำ Front-end
- Node.JS => ทำ Back-end
- MongoDB => ทำ Database
- Socket.io => ทำ Realtime-chat
- jsonwebtoken => ทำ Login
- bcrypt = เข้ารหัส Password
- Firebase Hosting => Deploy front-end
- Heroku => Deploy back-end
